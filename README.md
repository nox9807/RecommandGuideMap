## 🌟 간단한 소개
My Route는 **가까운 장소를 빠르게 찾고**,  
**테마별 인기 장소를 탐색하며**,  
**개인 루트를 저장하고 관리**할 수 있는 간편 지도 기반 길찾기 서비스입니다.

---

# 📌 주요 기능
- **홈** : 주변 추천 장소 탐색 및 길찾기
- **테마지도** : 카테고리 기반 장소 추천
- **즐겨찾기** : 내가 저장한 루트/장소 조회 및 루트 상세 확인
